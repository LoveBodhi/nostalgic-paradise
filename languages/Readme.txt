“pluralRules”是复数词规则，而非对应的语言翻译，因语言而异；分为：
"pluralRules" is for plural rule, instead of translation. Varied by language. They're:

1. singular-only

使用唯一的数词。
Only use unique singular form.

2. one-is-singular

只在数字为1的前提下的使用单数词，其余均使用复数词。
Uses plural form other than 1, which uses singular。

3. slavic

斯拉夫式复数词规则。个位数为1但非11的，使用单数词；个位数为2-4但非12-14的，使用双数词；其他情况使用复数词。
Slavic-style rules. Singular for a number end of 1, except 11; Dual form for a number end of 2-4, except 12-14; Plural for others.
